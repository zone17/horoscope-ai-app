Design Notes

  • Stars use amber/gold (#fbbf24) with glow — matches existing app accent color
  • Connecting lines at 35% opacity — subtle, not heavy
  • "Bright" stars (larger dots with stronger glow) mark key stars in each constellation
  • Selected state intensifies the glow (lighter gold #fde68a, stronger shadow)
  • Each icon is a simple SVG — 5-15 dots + lines, trivial to maintain
  • Renders identically across all platforms (no emoji variance)
  • Social captions still use Unicode emoji (platform-native in text)
  • Glassmorphism cards show how icons scale up for detail views
